
# Smart Attendance Management System (C)

A console-based attendance management system developed in C. 
This project allows users to add students, mark attendance, calculate attendance percentage, and identify students below the 75% threshold. 
The program uses file handling to store and retrieve data, ensuring persistence across runs.

## Features
- Add new students
- Mark daily attendance
- View individual attendance record
- Calculate attendance percentage
- Display students below 75% attendance
- File handling for data storage

## Concepts Used
- Structures
- Arrays
- Functions
- File Handling (Binary Files)
- Menu-driven Programming
